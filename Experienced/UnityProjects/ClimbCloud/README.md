
# CLIMB CLOUD

  

## The goal of this game
You need to climb the cloud and gets the banana and apple.
Also you have three lives avoid or shoot the monster and don't die!

## How to play
- Move : ← , →
- Jump : ↑
- Shoot : SpaceBar
